# Blue Envelope Program Website

A professional, data-driven digital presentation about the Blue Envelope Program and its benefits for implementation in Union County and North Carolina.

## Overview

This website showcases the Blue Envelope Program through comprehensive data visualizations, statistics, and implementation guidance. It focuses on demonstrating the benefits of local and statewide implementation using the program's signature blue and gold color scheme.

## Features

### Main Website (`index.html`)
- **Hero Section** with key statistics and call-to-action buttons
- **Interactive National Map** showing program adoption across states
- **North Carolina Focus** with comparative analysis and opportunity assessment
- **Union County Implementation Plan** with detailed timeline and costs
- **Comprehensive Benefits Analysis** with proven success stories
- **Professional Design** using Blue Envelope Program brand colors

### Training Materials (`training.html`)
- **Complete Officer Training Module** (15-minute roll-call format)
- **Visual Blue Envelope Recognition Guide** 
- **Autism-Informed Interaction Guidelines**
- **Interactive Training Scenarios** with step-by-step responses
- **Quick Reference Cards** for field use
- **Evidence-Based Best Practices**

## Key Data Highlighted

### National Statistics
- **15 states/territories** with Blue Envelope Programs
- **8 states** with statewide implementation
- **7 states** with local programs
- **2.21% of adults** have autism spectrum disorder
- **1 in 31 children** (3.2%) diagnosed with autism

### North Carolina Opportunity
- **#2 highest autism rate** nationally (1 in 39 children)
- **256,452** Union County population
- **6,411** estimated residents with autism
- **~$25,000-31,000** initial implementation cost
- **~$10,000-13,000** annual operating cost

### Law Enforcement Impact
- **20% of autistic individuals** have police contact by age 21
- **7x more likely** to interact with police
- **33-50%** of police fatalities involve disabilities
